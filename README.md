# Стэк
Laravel, LiveWire<br>
Tailwind CSS, Alpine JS

# Установка
```shell
git clone https://github.com/V1adJkee/afishka/; 
cd afishka;
composer install; 
npm install; npm run dev;
mv .env.example .env;
```
